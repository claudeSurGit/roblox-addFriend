# Roblox Clone

Un clone simple de Roblox avec une page de connexion.

## Installation

1. Installez Node.js depuis https://nodejs.org/

2. Ouvrez un terminal dans le dossier du projet et exécutez :
   ```
   npm install
   npm start
   ```

3. Ouvrez votre navigateur à http://localhost:3000

## Fonctionnalités

- Page de connexion qui sauvegarde les informations dans un fichier `logins.txt`

Pour consulter les informations de connexion, ouvrez le fichier `logins.txt` dans le dossier du projet.
